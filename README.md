# Subsets-Ruby

##### Problem: Find all the subsets of a given set.
```
Input: 

data = [a, b, c, d]


Output:

[null, null, null, null],

[a, null, null, null],

[null, b, null, null],

[null, null, c, null],

[null, null, null, d],

[a,b, null, null],

[a, null, c, null],

[a, null, null, d],

[null, b, c, null],

[null, b, null, d],

[null, null, c, d],

[a, b, c, null],

[a, b, null, d],

[a, null, c, d],

[null, b, c, d],

[a, b, c, d]
```
